# Overview

This is the **frontend interface** for the Task Manager web application, built using **React.js**.  
It connects to the Node.js + Express backend via REST APIs to allow users to:

- Register & Login securely  
- Manage tasks (Create, View, Delete)  
- Handle authentication via JWT tokens  
- Display responsive toast notifications for all actions  

---

# Features

User Registration & Login  
JWT-based Authentication  
Task CRUD (Create, Read, Delete)
Beautiful Gradient UI (Custom CSS) 
Logout Functionality
Success/Error Toast Notifications
Responsive Layout** (mobile-friendly)
